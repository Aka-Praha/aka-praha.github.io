# AKA Praha

Webové stránky Akademie klinické anatomie Praha

## Lokální vývoj

1. Nainstalujte Ruby (verze 3.1 nebo vyšší)
2. Nainstalujte závislosti: `bundle install`
3. Spusťte lokální server: `bundle exec jekyll serve`
4. Otevřete prohlížeč na `http://localhost:4000`

## Deployment

Stránky se automaticky buildují a nasazují na GitHub Pages pomocí GitHub Actions při každém push do větve `master`.
