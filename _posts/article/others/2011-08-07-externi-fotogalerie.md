Fotky z akcí sdílíme v externí fotogalerii na adrese **[http://linux.fjfi.cvut.cz/~aka/fotky/](http://linux.fjfi.cvut.cz/%7Eaka/fotky/)**.

Nahrávání a stahování

Fotky můžete nahrávat či stahovat jako soubory. Přístup přes SFTP běží serveru linux.fjfi.cvut.cz, uživatel aka s heslem, které se vám zde objeví po přihlášení: uid != 0)) { echo 'AKAgalerka'; } ?> .

Zde je návod, jak použít pro nahrávání a stahování fotek program WinSCP:

- Stáhněte si WinSCP z [této](http://winscp.net/eng/download.php) adresy a naistalujete na svůj počítač.
- Spustě WinSCP.
- V okně WinSCP login vyplňte následující údaje a klikněte na Login.

- Host name: linux.fjfi.cvut.cz
- User name: aka
- Password: uid != 0)) { echo 'AKAgalerka'; } ?>
- File protocol: SFTP

- Objeví se okno se dvěma panely, v pravo je váš počítač, nalevo naše fotogalerie (v adresáři fotky, nebo /home2/aka/webdav/fotky/).
- Nyní můžete pomocí klávesy F5 kopírovat soubory mezi vaším počítačem a fotogaleríí.
- V případě problémů napište, co nechodí a já vám odpovím (nejlépe asi do komentářů).

Udržujte pořádek :)

- Raději nepoužívete diakritiku ve jménech souborů a adresářů
- Všichni máme možnost cokoli smazat, odstraňujte prosím jen své omylem nahrané fotky.
- Fotky na server nahrávejte už protřiděné - těžko bude někdo obdivovat deset přeexponovaných fotek zadku zmítajícího se v převisu (neplatí pro holky ;) ).
- Pokud budeme chít po návratu z akce sdílet celé kvantum fotek, hodíme je dočasne do podadresáře.
- Fotky nezmenšujte, rádi se pokocháme originalni kvalitou.
- Na server určitě nepatří žádný nelegální materiál.
- Do podadresáře Osobní klidně nahrajte i hezké fotky ze skoukomých "alpinistických" akcí.
- Soukromé nesouvisející fotky by na serveru neměli ležet déle než, řekněme, dva týdny.
- Místa je zatím dost, ale přehánět to s objemem dat taky nemusíme.
- Lze sem nahrávat i autorská krátká videa.
- Autorství fotek vám odepřeno nebude, stačí do pole Artist EXIFu (tj. metadat obrázku) zapsat své jméno. To se pak zobrazí i na webu pod snímkem.
